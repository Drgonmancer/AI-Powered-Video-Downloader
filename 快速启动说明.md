# 快速启动说明

> 3步启动项目，最快30秒内跑通

---

## 方式一：一键启动（推荐）

```bash
python start.py
```

脚本会自动完成：环境检查 → 安装依赖 → 构建前端 → 启动服务 → 打开浏览器

---

## 方式二：开发模式（前后端分离调试）

### 步骤1：启动后端

```bash
cd backend
pip install -r requirements.txt
pip install yt-dlp
uvicorn main:app --reload --port 8976
```

启动后访问 http://localhost:8976/docs 可看到Swagger API文档。

### 步骤2：启动前端（如果已有前端代码）

```bash
cd frontend
npm install
npm run dev
```

前端开发服务器默认在 http://localhost:5173，已配置代理转发到后端。

---

## 验证后端是否正常工作

### 1. 健康检查

```bash
curl http://localhost:8976/api/health
# 返回: {"status":"ok"}
```

### 2. 测试解析接口（浏览器直接打开或用curl）

```bash
curl -X POST http://localhost:8976/api/parse \
  -H "Content-Type: application/json" \
  -d '{"url": "https://www.youtube.com/watch?v=dQw4w9WgXcQ"}'
```

### 3. 测试下载接口

```bash
curl -X POST http://localhost:8976/api/download \
  -H "Content-Type: application/json" \
  -d '{"url": "https://www.youtube.com/watch?v=dQw4w9WgXcQ"}'
```

返回 `{"task_id": "xxx", "status": "queued"}` 即成功创建任务。

### 4. 查看任务列表

```bash
curl http://localhost:8976/api/downloads
```

### 5. 查看设置

```bash
curl http://localhost:8976/api/settings
```
